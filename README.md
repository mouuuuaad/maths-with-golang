# MathsWithGolang

**MathsWithGolang** is a comprehensive Go library designed to implement algorithms across a wide range of mathematical domains. This project is structured for educational purposes and practical computation, covering everything from basic arithmetic in various number systems to advanced calculus and algebraic structures.

The project is organized into 9 main modules, each containing 8 chapters of detailed implementations:

*   **[01_Calculus](./01_Calculus)**: Fundamentals of calculus, limits, continuity, differentiation, integration, series, and multivariable calculus.
*   **[02_LinearAlgebra](./02_LinearAlgebra)**: Vectors, matrices, determinants, linear systems, eigenvalues, transformations, and decompositions.
*   **[03_Limits](./03_Limits)**: In-depth exploration of sequence and function limits, including L'H√¥pital's rule and continuity.
*   **[04_Derivatives](./04_Derivatives)**: Differentiation rules, higher-order derivatives, implicit/partial differentiation, and optimization.

*   **[06_NumericalSequences](./06_NumericalSequences)**: Arithmetic, geometric, and recursive sequences, convergence tests, and series.
*   **[07_AlgebraicStructures](./07_AlgebraicStructures)**: Abstract algebra concepts including groups, rings, fields, and polynomials.

## üöÄ Getting Started

### Prerequisites

*   Go 1.18 or higher

### Installation

Clone the repository:

```bash
git clone https://github.com/mouaadid/MathsWithGolang.git
cd MathsWithGolang
```

### Usage

To run the main demonstration program:

```bash
go run main.go
```

To run tests for a specific module (e.g., Calculus):

```bash
go test ./01_Calculus/...
```
