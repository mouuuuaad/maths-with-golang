# Module 14: Graph Theory

This module covers graph algorithms including traversal, shortest paths, minimum spanning trees, network flow, and classic graph problems.

## Chapters

1.  **Graph Basics**: Graph representation, BFS, DFS traversal algorithms.
2.  **Shortest Paths**: Dijkstra's algorithm, Floyd-Warshall all-pairs shortest path.
3.  **Topological Sort**: Kahn's algorithm, Cycle detection, Strongly Connected Components.
4.  **Minimum Spanning Trees**: Union-Find, Kruskal's algorithm, Prim's algorithm.
5.  **Advanced Pathfinding**: Bellman-Ford (negative weights), A* algorithm with heuristics.
6.  **Network Flow**: Ford-Fulkerson Max Flow, Min Cut algorithm.
7.  **Matching**: Bipartite Matching, Articulation Points, Bridges detection.
8.  **Classic Problems**: Eulerian Path, Hamiltonian Path, Graph Coloring.
