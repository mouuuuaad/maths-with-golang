# Module 03: Limits

This module provides numerical algorithms for evaluating mathematical limits of sequences and functions.

## Chapters

1.  **Sequence Limits**: $\lim_{n \to \infty} a_n$.
2.  **Function Limits**: $\lim_{x \to a} f(x)$.
3.  **One-Sided Limits**: Left ($x \to a^-$) and Right ($x \to a^+$) limits.
4.  **Limits at Infinity**: $\lim_{x \to \pm\infty} f(x)$.
5.  **Indeterminate Forms**: Handling $0/0$, $\infty/\infty$, etc.

7.  **Squeeze Theorem**: Verification of limits using bounding functions.
8.  **Continuity**: Checking continuity via limits.
