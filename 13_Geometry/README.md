# Module 13: Geometry

This module covers computational geometry including 2D/3D points, lines, shapes, transformations, coordinate systems, and intersection algorithms.

## Chapters

1.  **Points and Lines**: Point2D/3D, Distance, Midpoint, Slope, Line Equation, Point-to-Line distance.
2.  **2D Shapes**: Triangle (area, perimeter, centroid), Circle, Rectangle properties.
3.  **Transformations**: Rotation, Scaling, Translation, Reflection, Transformation matrices.
4.  **3D Geometry**: Vector3D operations (dot, cross, normalize), Plane, Line3D.
5.  **3D Solids**: Sphere, Cylinder, Cone, Box, Torus (volume, surface area).
6.  **Polygons**: Polygon area/perimeter, Centroid, Convexity test, Convex Hull algorithm.
7.  **Coordinate Systems**: Cartesian, Polar, Spherical, Cylindrical conversions.
8.  **Intersections**: Line-Line, Line-Circle, Circle-Circle, Point-in-Polygon algorithms.
