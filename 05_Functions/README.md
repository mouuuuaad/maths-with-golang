# Module 05: Functions

This module implements various mathematical functions, exploring their definitions, properties, and algorithmic implementations (e.g., via Taylor series).

## Chapters

1.  **Exponential**: $e^x$ and $a^x$.
2.  **Logarithm**: $\log_a(x)$ and $\log_{10}(x)$.
3.  **Natural Log**: $\ln(x)$.
4.  **Trigonometric**: $\sin, \cos, \tan$ and their inverses.
5.  **Hyperbolic**: $\sinh, \cosh, \tanh$.
6.  **Inverse**: Inverse functions and properties.
7.  **Primitives**: Antiderivatives and basic integration rules.
8.  **Special**: Gamma, Beta, and Bessel functions.
