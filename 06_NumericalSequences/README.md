# Module 06: Numerical Sequences

This module implements algorithms for generating and analyzing numerical sequences.

## Chapters

1.  **Arithmetic**: Sequences with constant difference.
2.  **Geometric**: Sequences with constant ratio.
3.  **Recursive**: Sequences defined by recurrence relations.
4.  **Convergence**: Tests for sequence convergence.
5.  **Monotone**: Checking monotonicity.
6.  **Bounded**: Checking bounds.
7.  **Series**: Summation of sequences.
8.  **Famous**: Fibonacci, Catalan, Lucas, etc.
