# Module 04: Derivatives

This module implements algorithms for numerical differentiation, optimization, and analysis of derivatives.

## Chapters

1.  **Definition**: Numerical derivative using finite differences.
2.  **Rules**: Implementation of differentiation rules (chain, product) for function composition.
3.  **Higher Order**: Second and third derivatives ($f'', f'''$).
4.  **Implicit**: Differentiation of implicit functions.
5.  **Partial**: Partial derivatives for multivariable functions.
6.  **Gradient**: Gradient vector, divergence, and curl.
7.  **Optimization**: Finding local minima/maxima (Gradient Descent, Newton's Method).
8.  **Taylor Series**: Taylor polynomial approximation.
