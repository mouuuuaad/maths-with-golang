# Module 11: Differential Equations

This module covers numerical methods for solving ordinary and partial differential equations, including stiff systems, adaptive methods, and physics applications.

## Chapters

1.  **First Order ODEs**: Euler, Heun, Midpoint, Runge-Kutta 4th order methods.
2.  **Systems of ODEs**: RK4 System solver, Euler System for coupled equations.
3.  **Adaptive Methods**: Adaptive RK45 with error control and step size adjustment.
4.  **Boundary Value Problems**: Shooting method, Finite Difference method.
5.  **Implicit Methods**: Verlet, Leapfrog, Backward Euler, Trapezoidal methods.
6.  **Stiff Equations**: Stiff solver, BDF-2, Crank-Nicolson implicit methods.
7.  **Partial Differential Equations**: Heat equation, Wave equation, Laplace equation.
8.  **Applications**: Simple Harmonic Oscillator, Damped Oscillator, Van der Pol, Lorenz attractor.
