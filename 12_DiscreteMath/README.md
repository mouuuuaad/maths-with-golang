# Module 12: Discrete Mathematics

This module covers number theory, set theory, relations, logic, combinatorics, recurrence relations, and graph theory fundamentals.

## Chapters

1.  **Number Theory**: GCD, LCM, Extended Euclidean, Modular Exponentiation, Modular Inverse.
2.  **Set Theory**: Set operations (Union, Intersection, Difference), Power Set, Subset.
3.  **Relations**: Reflexive, Symmetric, Transitive, Equivalence, Partial Order properties.
4.  **Logic**: Boolean operations (AND, OR, NOT, XOR, NAND, NOR), Truth Tables, Tautology.
5.  **Combinatorics**: Permutations, Combinations, Derangements, Stirling, Bell numbers.
6.  **Recurrence**: Fibonacci, Linear Recurrence, Partition numbers, Catalan numbers.
7.  **Basic Graphs**: Simple Graph, Degree, Connectivity, Euler, Bipartite, Chromatic number.
