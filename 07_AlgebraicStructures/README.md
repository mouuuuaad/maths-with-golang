# Module 07: Algebraic Structures

This module implements abstract algebraic structures and their properties.

## Chapters

1.  **Groups**: Group axioms, cyclic groups, permutations.
2.  **Subgroups**: Subgroups, cosets, Lagrange's theorem.
3.  **Rings**: Ring axioms, ideals, integral domains.
4.  **Fields**: Field axioms, finite fields.
5.  **Homomorphisms**: Structure-preserving maps.
6.  **Quotient**: Quotient groups and rings.
7.  **Polynomials**: Polynomial rings and arithmetic.
8.  **Applications**: Cryptography (RSA, Diffie-Hellman).
